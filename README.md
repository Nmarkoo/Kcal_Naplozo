# Kcal_Naplozo
A Fejlett Edzés &amp; Kalória Naplózó egy asztali alkalmazás, melynek célja az elvégzett fizikai aktivitások rögzítése és az elégetett kalóriák testsúly alapú, becsült kiszámítása. A projekt a PyQt6 keretrendszert használja a grafikus felhasználói felület (GUI) megvalósítására, biztosítva ezzel az intuitív és asztali környezetben is gyors adatbevitelt.
